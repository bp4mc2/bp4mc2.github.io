Classes

Act

The class of Acts. An act is a legal construct representing a set of activities, performed by an Actor and with respect to some Object.

URI:http://bp4mc2.org/def/skos-lex#Act
Subclass of:skos:Concept
Properties include:actor, agent, object, record
Inherited properties:dcterms:source
Used with property:affects, refersTo

Actor

The class of Actors. An actor is some person (individual, group or organisation) that performs some Act

URI:http://bp4mc2.org/def/skos-lex#Actor
Subclass of:skos:Concept
Has subclasses:Agent
Properties include:
Inherited properties:dcterms:source
Used with property:actor

Agent

The class of Agent. An agent is some person (individual, group or organisation) involved in some act, without actually performing the act.

URI:http://bp4mc2.org/def/skos-lex#Agent
Subclass of:Actor
Properties include:
Inherited properties:dcterms:source
Used with property:agent

Norm

The class of norms. A norm effects an Act. It implies what 'ought' or 'should not' be done with regard to an Act. A norm represents the set of rules that apply to an Act.

URI:http://bp4mc2.org/def/skos-lex#Norm
Properties include:affects, refersTo
See also:https://en.wikipedia.org/wiki/Norm_(philosophy), https://en.wikipedia.org/wiki/Theory_of_Legal_Norms

Object

The class of objects. An object is some thing (not an actor) that is created, used or changed during some Act

URI:http://bp4mc2.org/def/skos-lex#Object
Subclass of:skos:Concept
Has subclasses:Record
Properties include:
Inherited properties:dcterms:source
Used with property:object

Record

The class of records. A record is a some thing (not an actor) that is created to capture the execution of an act

URI:http://bp4mc2.org/def/skos-lex#Record
Subclass of:Object
Properties include:isAbout
Inherited properties:dcterms:source
Used with property:record

affects

Affects relates a norm with the Act that is affected by the norm. Such an act is restricted by the rules identified by the norm

URI:http://bp4mc2.org/def/skos-lex#affects

refersTo

RefersTo relates a norm with the Act that is refered by the norm. Such an act has some relation with the norm, without further specification. It might be effected by the norm, but could also be a precondition or simply mentioned in the rules.

URI:http://bp4mc2.org/def/skos-lex#refersTo
Property of:Norm
Subproperty of:skos:semanticRelation
Class of object:Act
